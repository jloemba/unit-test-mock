# Test unitaires

## Prérequis

Assurez-vous d'avoir python sur votre ordinateur. Pour vérifiez , executez la commande python -v

## Installation 
```
pip install unittest
pip install mock
```

## Lancement

Avec votre terminal , aller dans le repertoire /unit puis faites la commande suivante:

```
py exchange.test.py
```
